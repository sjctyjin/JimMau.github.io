曼陀羅九宮格思考法

曼陀羅九宮格思考法應用於專利發想

曼陀羅九宮格思考法是一種創意發想工具，適用於激發新的想法和概念。當應用於機構搭配影像技術的專利發想時，可以按照以下步驟：

1. 確定中心主題

將核心技術——機構搭配影像技術——放在九宮格的中心格，作為發想的出發點。

2. 定義八個探索方向

圍繞中心主題，確定八個相關的方面或領域：

  1. 新的應用領域：探索技術在新領域的應用潛力。
  2. 功能改進：提升現有功能，如影像解析度或機構精準度。
  3. 成本效益：降低成本或提高效率的方法。
  4. 用戶體驗：改善操作界面或使用體驗。
  5. 與其他技術整合：探索與AI、大數據分析等技術的整合潛力。
  6. 材料與製造：在材料選擇或製造工藝上的創新改進。
  7. 環境與可持續性：提高環境友好性或可持續發展性能。
  8. 法規與標準：符合或超越現行法規和標準的解決方案。

3. 產生具體想法

對每個方向產生具體的發想點或解決方案，例如，在”新的應用領域”中探索技術應用於自動駕駛系統的可能性。

4. 評估和選擇

評估每個想法的可行性、創新性和市場潛力，選擇最有前景的想法進行進一步開發。

5. 發展專利概念

深入分析選中的想法，確定其新穎性和非顯而易見性，開始撰寫專利描述和權利要求。

6. 準備專利申請

根據專利概念和撰寫的專利文件，準備和提交專利申請，並考慮聘請專利律師或專利代理人協助。

通過曼陀羅九宮格思考法，可以系統地探索技術應用，發掘新的專利發想，提高創新解決方案的質量和多樣性。

問題1. lxd init 時無法使用zfs

安裝lxd時發現無法使用zfs格式作為lxd storage pool的儲存格式，後發現zfs格式Xavier NX的核心架構arm不支援zfs格式。

原因 :

在筆電上使用VMWare建立的Ubuntu 20.04其核心是x86，因此在建立lxd init時預設是使用zfs格式，但此格式不影響lxc容器使用。

問題 2. lxc容器在Jetpack 4.6無法連接外網

建置 lxc ubuntu20.04容器後，發現無法連線外部網路，使用ping 時會出現 ping: socket: Operation not permitted 無法解決，

後斷定應該是該OS版本與lxc的支援相容性問題，因在ubuntu 20.04版本中不會有這個問題。

且eMMC僅有16GB，無法安裝太大的容量，因此須將OS系統安裝於M.2 SSD上。

問題 3. 將OS系統燒錄至SSD

將SSD安裝上Xavier NX後，需進行磁碟分割，規劃磁區後，將Xavier NX透過micro USB 連接電腦後，使用Nvidia SDK Manager 進行版本移植
