二、 安裝依賴套件

安裝完成後，執行指令cd yolov5 至資料夾下

執行pip安裝依賴套件。

pip install -r requirements.txt

若pytorch pip安裝無錯誤可跳過以下步驟。

注 : 若pytorch pip安裝失敗則需手動安裝，若無錯誤可跳過以下步驟。

至 Previous PyTorch Versions| PyTorch 下載對應自己電腦GPU版本的pytorch

先在cmd終端下輸入 nvidia-smi 查詢電腦GPU CUDA Version資訊。

以本機電腦為例: CUDA Version 最高支援為11.7，所以要找支援11.7以下的pytorch版本，從官網下載支援的頁面找到v1.12.1版本，複製下方CUDA 11.6的安裝指令(分為conda與wheel安裝方式)，若使用Anaconda則使用conda安裝，若使用pip安裝則為下方的pip安裝指令。

注:

1.若無GPU可安裝CPU only的安裝指令

2.torch的版本與torchvision版本是有相對應的，不可安裝錯誤。

3.安裝GPU版本torch前請先將CUDA環境安裝設定完成。
